# Page

`vgr-page` är rotelement, som rymmer hela sidans innehåll.

## Användning

Import it via:

```html
<vgr-page>
  <vgr-page-header>
  </vgr-page-header>
  <vgr-page-body>
    <vgr-page-block></vgr-page-block>
    <vgr-action-panel></vgr-action-panel>
    <vgr-grid></vgr-grid>
  </vgr-page-body>
</vgr-page>
```
