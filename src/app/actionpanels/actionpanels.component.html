Översikt

<vgr-action-panel> är en panel som kan öppnas och stängas. För mer information, se dokumentation för sidhantering. Öppna och stäng panelen högst upp på denna sida med knappen nedan:


Stäng panel 1 Öppna panel 1

Stäng panel 2 Öppna panel 2

Förändras med tema: nej

API
Namn Beskrivning Exempel
title: string Anger action panelens titel. [title]="'En titel'"
expanded: boolean Anger om panelen ska vara öppen eller stängd. [expanded]="true"
showCloseButton: boolean Anger huruvida en stänga-knapp ska visas i övre högra hörnet av panelen. Default är false. [showCloseButton]="true"
