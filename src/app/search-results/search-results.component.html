<vgr-search-result>


En sökresultat-lista som går att koppla till en input-komponent. För att den skall placera sig rätt behöver man wrappa båda komponenterna i en div med klassen search-result-wrapper. Klicka i input-fältet för att visa ett exempel på sökresultatlistan.



Förändras med tema: nej

Exempel på sökning 1

Exempel på sökning 2

      <div class="search-result-wrapper">
        <vgr-filter-textbox (valueChanged)="filterSearch($event)" [value]="filterBoxValue"></vgr-filter-textbox>
        <vgr-search-result [items]="filteredItems" [visible]="dropdownVisible" (itemClick)="click($event)" [description]="searchDescription"></vgr-search-result>   
      </div>
Namn Beskrivning Exempel
description : string En text som visas ovanför sökträffarna. [description]="55 träffar i "VGR" KIV."
noResultsText : string en text som visas om inga sökträffar hittas. [noResultsText]="Inga sökträffar hittades."
items : array Listan på sökträffar i rätt format. [items]="['{{'{'}}value: "123", displayName: " Min mottagning"{{'}'}}, ...]"
maxItems : number Antalet sökträffar som maximalt skall visas i listan. 25st är default. [maxItems]="25"
visible : boolean Styr hurvida listanvisas eller ej. [visible]="isListVisible"
itemClick : event Event som triggas när man klickar på en listitem, skickar med datan för det listitemet. (itemClick)="doSomething($event)"
