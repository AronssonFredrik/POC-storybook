<vgr-icon>

Det finns nu stöd för att använda Font Awsome (free). Genom att använda komponenten vgr-icon kan dessa ikoner användas https://fontawesome.com/icons?d=gallery&m=free.


Komponenten är en anpassad variant av Font Awsomes angular komponent och stöder dens Basic funktionalitet angular-fontawesome features (basic). Dok måste måste implementationen följa api nedan.

Namn Beskrivning Exempel
icon: string Ikonens textkod. Möjliga värden är koder enligt Font Awsomes dokumetation för gratisikoner. Default inget värde. [icon]="'star'"
solid: boolean Sätter om ikonen skall vara solid. Möjliga värden är true (solid) eller false (regular). Default är true. [solid]="false"
size: string Ikonens storlek. Möjliga värden är 'xs' (.75em), 'sm' (.875em), 'lg' (1.33em), '[n]x' ([n]em) etc där [n] är ett tal ex '2x' (2em). Default tom sträng (följer fontstorleken). [size]="'xs'"
color: string Sätter ikonens färg. Möjliga värden är 'primary' (följer temat), 'text' (följer textfärgen), 'success', 'error'. Defaultvärde är 'primary'. [color]="'text'"
disabled: boolean Visar ikon i som inaktiv. Default är false. [disabled]="true"
focusable: boolean Styr om ikonen skall vara fokuserbar. Default är false. [focusable]="true"
ariaLabel: string Möjlighet att sätta en eget röstmeddelande på ikonens area-label. Default tom sträng . [ariaLabel]="'min favorit'"
fixedWidth: boolean Låser storleken till ett fast värde. Default är false. [fixedWidth]="true"
rotate: number Roterar ikonen i antal grader. Tillåtna värden är 90, 180 och 270. Default är 0. [rotate]="90"
flip: string Spegelvänder ikonen. Tillåtna värden är 'horizontal', 'vertical' och 'both'. Default är tom sträng. [flip]="'vertical'"
spin: bolean Gör att ikonen spinner. Tillåtna värden är true eller false. Default är false. [spin]="true"
pulse: bolean Gör att ikonen spinner på ett snabbare och pulserande sätt. Tillåtna värden är true eller false. Default är false. [pulse]="true"
border: bolean Lägger på en tunn border runt ikonen. Tillåtna värden är true eller false. Default är false. [border]="true"
pull: string Skuter ikonen åt vänster eller höger. Tillåtna värden är 'left' och 'right'. Default är tom sträng. [pull]="'right'"
